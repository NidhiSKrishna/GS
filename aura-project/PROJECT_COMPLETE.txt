🚀 HOW TO RUN
=============

QUICK START (Windows):
1. Double-click: start.bat
   (Opens two terminal windows automatically)

QUICK START (Mac/Linux):
1. Open terminal and run: bash start.sh
   (Starts both servers in background)

MANUAL START:
1. Terminal 1: cd backend && npm start
   → Backend runs on http://localhost:5000
2. Terminal 2: cd frontend && npm run dev
   → Frontend runs on http://localhost:5173

DEVELOPMENT MODE:
1. Terminal 1: cd backend && npm run dev
   → Backend with auto-restart on changes
2. Terminal 2: cd frontend && npm run dev
   → Frontend with Vite HMR (hot reload)

🔧 CUSTOMIZATION GUIDE
======================

Change Colors:
1. Edit: frontend/tailwind.config.js
2. Update theme colors
3. Rebuild with: npm run build

Change Port:
1. Edit: backend/server.js (line 11)
2. Update: frontend/src/pages/Detect.jsx
3. Search: "http://localhost:5000"
4. Replace with new port

Add New Pages:
1. Create: frontend/src/pages/NewPage.jsx
2. Import in: frontend/src/App.jsx
3. Add route: <Route path="/newpage" element={<NewPage />} />
4. Add nav link

Add Backend Routes:
1. Edit: backend/server.js
2. Add: app.post('/new-endpoint', (req, res) => { ... })
3. Restart backend

📞 TROUBLESHOOTING QUICK REFERENCE
====================================

Issue: "Cannot find module..."
Fix: Run npm install in that directory

Issue: "EADDRINUSE: Port already in use"
Fix: Kill process on port 5000 or change port

Issue: Frontend shows blank page
Fix: Check browser console (F12), refresh page, check if backend is running

Issue: API calls failing
Fix: Verify backend is running, check CORS, verify URL in Detect.jsx

Issue: Styles not loading
Fix: Run npm install in frontend, restart dev server

Issue: Build errors
Fix: Delete node_modules, delete package-lock.json, run npm install
