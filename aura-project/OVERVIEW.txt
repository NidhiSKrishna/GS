📊 PROJECT OVERVIEW

   A modern full-stack web application for detecting deepfakes and identity
   threats using AI analysis with a beautiful dark cybersecurity UI.

🔌 API DOCUMENTATION

   Health Check:
   GET http://localhost:5000/
   
   Response:
   {
     "message": "AURA Backend is running",
     "status": "ok",
     "version": "1.0.0"
   }

   Deepfake Scan:
   POST http://localhost:5000/scan
   Body: { "image": "base64-image-data" }
   
   Response:
   {
     "match": "82%",
     "deepfake": "Likely",
     "risk": "High",
     "timestamp": "2026-04-18T10:30:00.000Z",
     "confidence": "0.95"
   }
