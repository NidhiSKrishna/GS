═══════════════════════════════════════════════════════════════════════════════
                      HOW TO RUN THE PROJECT
═══════════════════════════════════════════════════════════════════════════════

OPTION 1: Windows Batch Script (Easiest)
  1. Double-click: start.bat
  2. Opens two terminals automatically
  3. Backend runs on port 5000
  4. Frontend runs on port 5173

OPTION 2: Unix/Mac Shell Script
  1. Open terminal
  2. Run: bash start.sh
  3. Both servers start in background

OPTION 3: Manual (Any OS)
  Terminal 1:
    cd backend
    npm start
  
  Terminal 2:
    cd frontend
    npm run dev

THEN:
  1. Open browser: http://localhost:5173
  2. Explore Dashboard
  3. Try Deepfake Detection
  4. Click "Scan Image" to test API

                      CUSTOMIZATION GUIDE
═══════════════════════════════════════════════════════════════════════════════

Change Backend Port:
  Edit: backend/server.js
  Line: const PORT = 5000;
  Update: frontend/src/pages/Detect.jsx axios URL

Change Colors/Theme:
  Edit: frontend/tailwind.config.js
  Modify: theme.extend.colors

Add New Pages:
  1. Create: frontend/src/pages/NewPage.jsx
  2. Import in: frontend/src/App.jsx
  3. Add: <Route path="/page" element={<NewPage />} />

Add Backend Routes:

  Add: app.post('/endpoint', (req, res) => { ... })

                      QUICK TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════

Issue: Port 5000 Already in Use
  Fix: Kill process or change port in server.js

Issue: Dependencies Not Installed
  Fix: cd backend && npm install && cd ../frontend && npm install

Issue: Frontend Shows Blank Page
  Fix: Check console (F12), refresh page, ensure backend running

Issue: API Calls Failing
  Fix: Verify backend running, check CORS, check API URL

Issue: Styles Not Applying
  Fix: Restart dev server, clear browser cache

Issue: Module Not Found
  Fix: npm install in that directory
