STEP 1: START THE SERVERS
================================================================================

WINDOWS:
  Double-click: start.bat

MAC/LINUX:
  Run: bash start.sh

MANUAL:
  Terminal 1: cd backend && npm start
  Terminal 2: cd frontend && npm run dev


STEP 2: OPEN IN BROWSER
================================================================================

  Go to: http://localhost:5173

  You should see the AURA interface with:
  • Dark cybersecurity theme
  • Navigation bar with Dashboard and Detect links
  • "Dashboard" link to view status cards
  • "Detect" link to test deepfake scanning


STEP 3: EXPLORE FEATURES
================================================================================

Dashboard Page:
  • Shows 3 info cards: Identity Status, Recent Scan, Risk Score
  • Cards have smooth animations on hover
  • Displays system information

Detect Page:
  • Click "Scan Image" button
  • Wait for the scan to complete
  • See results with:
    - Match % (confidence)
    - Deepfake Detection (Likely/Not Likely)
    - Risk Level (Low/Medium/High)
    - Color-coded display (Green/Yellow/Red)

================================================================================
                      TROUBLESHOOTING
================================================================================

Issue: Blank page in browser
  → Check browser console (F12)
  → Ensure backend is running (check terminal)
  → Try refreshing the page

Issue: API not connecting
  → Verify backend is running on port 5000
  → Check if CORS errors in browser console
  → Verify frontend API URL

Issue: Port 5000 already in use
  → Kill process on port 5000
  → Or change port in backend/server.js

Issue: Dependencies not installed
  → cd backend && npm install
  → cd frontend && npm install
